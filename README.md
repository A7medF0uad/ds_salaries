# Data Science Salaries — ML Pipeline & Dashboard

Predicts data science salaries (USD) from job/company attributes using a
leakage-safe RandomForest pipeline, with an interactive Streamlit dashboard
for exploration and live predictions.

## Project structure

```
ds-salaries-prediction/
├── data/
│   └── ds_salaries.csv          # not committed — see "Data" below
├── model/                        # generated by train_model.py
│   ├── salary_model.pkl
│   ├── feature_options.pkl
│   └── feature_importances.csv
├── notebooks/
│   └── salary_prediction.ipynb   # original analysis notebook
├── train_model.py                # trains + saves the model
├── app.py                        # Streamlit dashboard
├── requirements.txt
├── README.md

```

## Data

Expects `data/ds_salaries.csv` with columns:
`work_year, experience_level, employment_type, job_title, salary,
salary_currency, salary_in_usd, employee_residence, remote_ratio,
company_location`.

## Setup

```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## 1. Train the model

```bash
python train_model.py
```

This:
- Splits the data (60/20/20 train/val/test) and reports honest RMSE/R² on the held-out test set.
- Bundles rare categories (< 5 occurrences) into `"Other"`, learned **from training data only** to avoid leakage.
- Keeps `experience_level` as a numeric ordinal (1–4) instead of one-hot encoding it.
- Log-transforms the target (`TransformedTargetRegressor`) to handle salary skew.
- Retrains a final production model on the **full dataset** and saves it to `model/salary_model.pkl` for the dashboard.

## 2. Run the dashboard

```bash
streamlit run app.py
```

Two tabs:
- **Data Explorer** — filterable charts: salary distribution, salary by experience level, salary by remote ratio, top-paying job titles, and model feature importance.
- **Salary Predictor** — pick job title, experience, location, etc., and get a live salary estimate from the trained model.

#### View it in: [ds-salaries-prediction](https://ds-salaries-prediction.streamlit.app/)

## Model performance (held-out test set)

| Metric | Value |
|---|---|
| RMSE | ~$50,000 (see console output from `train_model.py` for your exact run) |
| R² | ~0.36 |

Predictions are estimates based on historical patterns, not precise figures — the dashboard surfaces the test R²/RMSE next to the prediction so it's clear how much uncertainty to expect.

## Key findings

- `employee_residence` and `job_title` are the strongest salary drivers.
- Fully remote roles show the highest average salary in this dataset; hybrid roles the lowest.
- Salary rises with experience level, though with wide variance at senior/mid levels.
